# Nussy Dev

Official website for **Nussy Co.** — a web agency that builds websites for others.

## Stack

- Pure HTML, CSS, JavaScript — no dependencies
- Google Fonts: Space Grotesk + Space Mono
- Ready for GitHub Pages (zero build step)

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<username>.github.io/<repo-name>`

## Sections

- Hero with animated text + marquee
- About / stats counters
- Services grid
- Projects portfolio
- CTA band
- Contact form
- Custom cursor + scroll reveal animations
